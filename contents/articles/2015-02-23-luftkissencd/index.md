Letztes Wochenende habe ich mit B. ein einfaches Luftkissen-Fahrzeug aus einer leeren Plastikflasche, einer CD und ein paar Luftballons gebastelt:
die Luft im Ballon strömt durch das Loch unter die CD bildet so ein Kissen. 

Man kann den Luftballon vermutlich auch direkt oder mit einer Garnspule oder so auf der CD befestigen, aber diese Variante gefällt mir besser,
weil der Ballon leichter nachgeladen werden kann: einfach aufpusten und auf den Deckel schrauben.


- |image|luftkissen_cd_hals.jpg|Flaschenhals abschneiden|Brutal: 11-jähriger schneidet Flasche den Hals ab.|
- |image|luftkissen_cd_deckel.jpg|Flaschendeckel durchbohren|Ein kleineres Loch hätte vermutlich auch gereicht - dann hält eine Luftballon-Ladung länger.|
- |image|luftkissen_cd_klebe.jpg|Deckel mittig auf CD kleben|Den Deckel mittig auf die CD kleben, so dass das Loch im Deckel über dem Loch in der CD ist.
    Vermutlich hätte auch jede andere Art von Kleber funktioniert, aber ich hatte nun mal gerade Heißklebe parat.|
- |image|luftkissen_cd_ballon_s.jpg|Ballon auf Flaschenhals ziehen|Der Ballon wird auf der Seite auf den Flaschenhals gezogen, die kein Gewinde hat.|
- |image|luftkissen_cd_ballon_l.jpg|Ballon aufpusten|Wenn der Ballon gut gefüllt ist, kann man ihn zuhalten und auf den Flaschendeckel schrauben.|
- |image|luftkissen_cd_final.jpg|Er schwebt|Gut, auf einem Foto wirkt das vielleicht etwas unspektakulär, aber in Echt, also ich kann dir sagen.
    <br>Erinnert sich eigentlich noch jemand an AirHockey?|

Die Idee dazu habe ich schon recht häufig gesehen, aber die Variante bei 
    [Instructables](http://www.instructables.com/id/CD-Hovercraft-for-tabletop-hockey/) hat mir am besten gefallen.
