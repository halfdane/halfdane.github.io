Luftkissen-CD

Letztes Wochenende habe ich mit B. ein einfaches Luftkissen-Fahrzeug aus einer leeren Plastikflasche, einer CD und ein paar Luftballons gebastelt: die Luft im Ballon strömt durch das Loch unter die CD bildet so ein Kissen.

Man kann den Luftballon vermutlich auch direkt oder mit einer Garnspule oder so auf der CD befestigen, aber diese Variante gefällt mir besser, weil der Ballon leichter nachgeladen werden kann: einfach aufpusten und auf den Deckel schrauben.

Die Idee dazu habe ich schon recht häufig gesehen, aber die Variante bei Instructables hat mir am besten gefallen.
